# Скачать датасет

[Назад к списку компонентов](../README.md)

## Назначение

Компонент скачивает и, по необходимости, распаковывает архив с датасетами, приложенными к соревнованию Kaggle.

## Входные порты

| Название                | Тип        |
|:------------------------|:-----------|
| Входные переменные      | Переменные |

### Переменные в порте "Входные переменные"

| №  | Метка                         | Тип                                     | Значение по умолчанию  |
|:---|:------------------------------|:----------------------------------------|:-----------------------|
| 1  | Идентификатор соревнования    | ![](./img/string.svg) Строковый         | null                   |
| 2  | Расположение                  | ![](./img/string.svg) Строковый         | null                   |
| 3  | Распаковать ZIP архив         | ![](./img/logical.svg) Логический       | true                   |

1. **Идентификатор соревнования** — имя, указанное в URL. Например `home-credit-default-risk` в https://www.kaggle.com/competitions/home-credit-default-risk.
2. **Расположение** — директория, в которую будет скачан архив с датасетами соревнования. Если переменная пустая, то в архив будет скачан в директорию пакета с созданием папки с именем соревнования.
3. **Распаковать ZIP архив** — если флаг `true`, то архив будет распакован в отдельную папку с этим же именем. Архив при этом не удаляется.
